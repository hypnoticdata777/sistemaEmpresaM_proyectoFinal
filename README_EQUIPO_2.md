# Equipo 2 - Compras y proveedores

Este proyecto conserva la estructura general indicada en el PDF, pero solo contiene codigo funcional del Equipo 2.

## Paquete del equipo

```text
com.empresa.compras
```

## Modulos del Equipo 2

```text
proveedor
compra
detallecompra
```

Cada modulo incluye:

- Model
- Repository
- Service
- Controller

## JSON del Equipo 2

Consume:

```text
importaciones/productos.json
```

Produce:

```text
exportaciones/entradas_inventario.json
```

Clases JSON colocadas segun la estructura del PDF:

```text
com.empresa.json.dto
com.empresa.json.importador
com.empresa.json.exportador
```
